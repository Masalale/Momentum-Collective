## 🧞 Commands

All commands are run from the root of the project:

| Command           | Action                                           |
| :---------------- | :----------------------------------------------- |
| `bun install`     | Installs dependencies                            |
| `bun run dev`     | Starts local dev server at `localhost:4321`      |
| `bun run build`   | Build your production site to `./dist/`          |
| `bun run preview` | Preview your build locally, before deploying     |

## 🚀 Development

Start the development server:

```bash
bun run dev
```

Then visit: http://localhost:4321

The site will automatically reload when you make changes to any files.

## 📦 Build & Deploy

### Build for production:
```bash
bun run build
```

This creates an optimized static site in the `dist/` directory.

### Deploy Options:
- **Vercel**: Connect your Git repo for automatic deployments
- **Netlify**: Drag & drop the `dist/` folder
- **GitHub Pages**: Use the provided workflow in MIGRATION_GUIDE.md
